hexo基础命令

序言

我经常忘记hexo的命名，不得不去官网找教程，就是这个地址：https://hexo.io/zh-cn/docs/。这里简单记录几个命令和常用的写作技巧

命令篇

  1. hexo init [folder文件夹名]，创建一个项目/网站/站点，如果没有文件夹名，Hexo 默认在目前的文件夹建立网站。
  2. hexo new [layout] <title>，新建文章，layout是指分类的意思，常用的有‘page’、‘draft’，指的是分页和草稿，没有就是指post文章，默认发到首页
  3. hexo g，生成网页文件，如果配上d，即hexo g -d，就会生成后，立即部署。
  4. hexo s，启动服务器，预览发布结果
  5. hexo d，部署到服务器。
  6. hexo clean，清除缓存文件 (db.json) 和已生成的静态文件 (public)。尤其是更换主题后，无论如何也不生效，就用这个清理
  7. hexo list，列出网站的资料

写作篇

  1. 写作的内容格式都遵循markdown的形式，主要是学会md语法即可
  2. 文章存放的路径遵守基本的方法就行
  3. 善用标签分类，文章要打好标签，方便日后整理
